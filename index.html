Installation Steps

  1. Download the extension ZIP file using the button above
  2. Extract the ZIP file to a folder on your computer
  3. Open Chrome and navigate to chrome://extensions/
  4. Enable Developer mode (toggle in top-right corner)
  5. Click "Load unpacked" and select the extracted frontend/ folder
  6. Pin the extension icon to your toolbar for easy access
  7. Select any text on a webpage and click the extension icon!

Requirements

  • Google Chrome Browser (version 88 or higher)
  • Internet connection (for AI processing)
  • No API key needed - backend is deployed and ready!
